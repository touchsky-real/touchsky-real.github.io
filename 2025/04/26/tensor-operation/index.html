|жөҸи§ҲйҮҸ:

е®ҡд№ү

еј йҮҸ(tensor)жҳҜдёҖз»„е…·жңүеӨҡз»ҙеәҰз»“жһ„зҡ„ж•°гҖӮз»ҙеәҰзҡ„дёӘж•°з§°дёә йҳ¶(rank),йҖҡиҝҮtensor.dim()жҹҘзңӢгҖӮеј йҮҸзҡ„е…·дҪ“еҪўзҠ¶йҖҡиҝҮtensor.shape()жҹҘзңӢгҖӮ

зҙўеј•ж–№ејҸ

еҲҮзүҮзҙўеј•

еғҸ python зҡ„ list е’Ң numpy зҡ„ array дёҖж ·пјҢеј йҮҸеҸҜд»ҘдҪҝз”Ё start:stop жҲ– start:stop:step иҝҷж ·зҡ„иҜӯжі•иҝӣиЎҢеҲҮзүҮгҖӮstop жҳҜдёҚиў«еҢ…жӢ¬еңЁеҶ…зҡ„з¬¬дёҖдёӘе…ғзҙ гҖӮ

и®ҝй—®еј йҮҸдёӯзҡ„еҚ•иЎҢжҲ–еҚ•еҲ—жңүдёӨз§Қеёёи§Ғж–№жі•пјҡ

  • дҪҝз”ЁдёҖдёӘж•ҙж•°зҙўеј•дјҡдҪҝеј йҮҸзҡ„йҳ¶пјҲrankпјүеҮҸе°‘ 1гҖӮ
  • дҪҝз”ЁдёҖдёӘй•ҝеәҰдёә 1 зҡ„еҲҮзүҮпјҲsliceпјүеҲҷдҝқжҢҒеј йҮҸзҡ„йҳ¶дёҚеҸҳгҖӮ

ж•ҙж•°еј йҮҸзҙўеј•

ж•ҙж•°зҙўеј•еҸҜд»Ҙдә’жҚўеј йҮҸиЎҢе’ҢеҲ—зҡ„йЎәеәҸгҖӮ

жӣҙдёҖиҲ¬ең°иҜҙпјҢз»ҷе®ҡдёӨдёӘзҙўеј•ж•°з»„ idx0 е’Ң idx1пјҢе®ғд»¬еҗ„иҮӘжңү N дёӘе…ғзҙ пјҢйӮЈд№ҲиЎЁиҫҫејҸ a[idx0, idx1] зӯүд»·дәҺпјҡ

еёғе°”еј йҮҸзҙўеј•

еёғе°”еј йҮҸзҙўеј•е…Ғи®ёдҪ ж №жҚ®дёҖдёӘеёғе°”жҺ©з ҒпјҲboolean maskпјүйҖүеҸ–еј йҮҸдёӯд»»ж„Ҹзҡ„е…ғзҙ гҖӮ

еј йҮҸж“ҚдҪң

жҢүе…ғзҙ ж“ҚдҪң

еҹәжң¬зҡ„ж•°еӯҰеҮҪж•°еңЁеј йҮҸдёҠжҳҜжҢүе…ғзҙ пјҲelementwiseпјүж“ҚдҪңзҡ„пјҢеҸҜд»ҘйҖҡиҝҮиҝҗз®—з¬ҰйҮҚиҪҪ,torch жЁЎеқ—дёӯзҡ„еҮҪж•°пјҢд»ҘеҸҠеј йҮҸеҜ№иұЎзҡ„ж–№жі•жқҘдҪҝз”ЁгҖӮ

reduction ж“ҚдҪң

жҲ‘д»¬жңүж—¶еёҢжңӣеҜ№еј йҮҸзҡ„йғЁеҲҶжҲ–е…ЁйғЁе…ғзҙ жү§иЎҢиҒҡеҗҲж“ҚдҪңпјҢжҜ”еҰӮжұӮе’Ң,иҝҷзұ»ж“ҚдҪңиў«з§°дёә reduction ж“ҚдҪңгҖӮreduction ж“ҚдҪңеҢ…жӢ¬жұӮе’Ң(sum)гҖҒжұӮе№іеқҮеҖј(mean)гҖҒеҸ–жңҖе°ҸеҖј(min)гҖҒеҸ–жңҖеӨ§еҖј(max)зӯүгҖӮ

reduction ж“ҚдҪңйҷҚдҪҺеј йҮҸзҡ„йҳ¶гҖӮдј е…ҘеҸӮж•° dim=d еҗҺпјҢиҫ“е…Ҙеј йҮҸдёӯзҙўеј•дёә d зҡ„з»ҙеәҰдјҡд»Һиҫ“еҮәеј йҮҸзҡ„еҪўзҠ¶дёӯз§»йҷӨгҖӮеҰӮжһңдј е…ҘеҸӮж•° keepdim=TrueпјҢеҲҷжҢҮе®ҡзҡ„з»ҙеәҰдёҚдјҡиў«з§»йҷӨиҖҢжҳҜеҸҳдёә 1гҖӮеңЁиҝӣиЎҢ reduction ж“ҚдҪңж—¶пјҢиҖғиҷ‘еј йҮҸзҡ„еҪўзҠ¶жҜ”иҫғдҫҝдәҺзҗҶи§ЈгҖӮ

жңүзҡ„ reduction ж“ҚдҪңиҝ”еӣһдёҚжӯўдёҖдёӘеҖјпјҢжҜ”еҰӮ min дјҡиҝ”еӣһжҢҮе®ҡз»ҙеәҰдёҠзҡ„жңҖе°ҸеҖјпјҢд»ҘеҸҠжңҖе°ҸеҖјжүҖеңЁзҡ„дҪҚзҪ®зҙўеј•пјҡ

д»ҘдёҠзҡ„torch.minиҫ“е…Ҙзҡ„еҸӮж•°дёҖиҲ¬жҳҜдёҖдёӘеј йҮҸгҖӮжңүеҸҰдёҖдёӘеҮҪж•°torch.minimumз”ЁдәҺжҢүе…ғзҙ жҜ”иҫғдёӨдёӘ tensorпјҢиҝ”еӣһеҜ№еә”е…ғзҙ дёӯиҫғе°Ҹзҡ„йӮЈдёӘгҖӮзұ»дјјзҡ„иҝҳжңүtorch.maximumгҖӮ

Reshaping ж“ҚдҪң

view

PyTorch жҸҗдҫӣдәҶеӨҡз§Қж“ҚдҪңеј йҮҸеҪўзҠ¶зҡ„ж–№жі•гҖӮжҜ”еҰӮview()иҝ”еӣһдёҖдёӘдёҺиҫ“е…Ҙе…·жңүзӣёеҗҢе…ғзҙ ж•°йҮҸпјҢдҪҶеҪўзҠ¶дёҚеҗҢзҡ„ж–°еј йҮҸгҖӮ

view() жҢүз…§еҺҹеј йҮҸзҡ„иЎҢдјҳе…ҲйЎәеәҸжқҘйҮҚж–°жҺ’еҲ—е…ғзҙ пјҢиҖҢеј йҮҸеңЁеҶ…еӯҳдёӯеӯҳеӮЁд№ҹжҳҜиЎҢдјҳе…Ҳзҡ„гҖӮ

жӯЈеҰӮе…¶еҗҚз§°жүҖжҡ—зӨәпјҢйҖҡиҝҮview()иҝ”еӣһзҡ„еј йҮҸдёҺиҫ“е…Ҙе…ұдә«зӣёеҗҢзҡ„ж•°жҚ®пјҢеӣ жӯӨеҜ№е…¶дёӯдёҖдёӘзҡ„жӣҙж”№е°ҶеҪұе“ҚеҸҰдёҖдёӘгҖӮ

жҲ‘д»¬еҸҜд»ҘдҪҝз”Ёview()е°Ҷзҹ©йҳөеұ•е№ідёәеҗ‘йҮҸпјҢе°ҶдёҖз»ҙеҗ‘йҮҸиҪ¬жҚўдёәдәҢз»ҙзҡ„иЎҢзҹ©йҳөжҲ–еҲ—зҹ©йҳөгҖӮ

unsqueeze

unsqueezeеңЁжҢҮе®ҡдҪҚзҪ®жҸ’е…ҘдёҖдёӘз»ҙеәҰ,е®ғжңүдёҖз§Қз®ҖдҫҝеҶҷжі•гҖӮ
a[:, None]зҡ„ж„ҸжҖқжҳҜеңЁеҺҹеј йҮҸзҡ„еҗҺйқўж–°еўһдёҖдёӘз»ҙеәҰгҖӮ

cat дёҺ stack

torch.cat жІҝзҺ°жңүзҡ„з»ҙеәҰжӢјжҺҘеј йҮҸпјҢдёҚдјҡеўһеҠ ж–°зҡ„з»ҙеәҰгҖӮ
еҰӮжһңдҪ жңүдёӨдёӘеҪўзҠ¶дёә (2, 3) зҡ„еј йҮҸпјҢдҪҝз”Ё torch.cat жІҝзқҖз¬¬ 0 з»ҙпјҲиЎҢпјүжӢјжҺҘеҗҺпјҢз»“жһңеј йҮҸзҡ„еҪўзҠ¶е°ҶжҳҜ (4, 3)

torch.stack жІҝдёҖдёӘж–°зҡ„з»ҙеәҰе ҶеҸ еј йҮҸпјҢдјҡеўһеҠ дёҖдёӘж–°зҡ„з»ҙеәҰгҖӮ
еҰӮжһңдҪ жңүдёӨдёӘеҪўзҠ¶дёә (2, 3) зҡ„еј йҮҸпјҢдҪҝз”Ё torch.stack жІҝж–°зҡ„з¬¬ 0 з»ҙе ҶеҸ з»“жһңеј йҮҸзҡ„еҪўзҠ¶е°ҶжҳҜ (2, 2, 3)

еј йҮҸдёҺж ҮйҮҸзҡ„и®Ўз®—

еј йҮҸе’Ңж ҮйҮҸпјҲScalarпјүеҸҜд»ҘзӣҙжҺҘиҝӣиЎҢиҝҗз®—пјҢиҖҢдё”ж“ҚдҪңжҳҜе…ғзҙ зә§пјҲelement-wiseпјүзҡ„пјҢPyTorch дјҡиҮӘеҠЁе№ҝж’ӯж ҮйҮҸеҲ°еј йҮҸзҡ„жҜҸдёӘе…ғзҙ гҖӮжҜ”еҰӮ+гҖҒ-гҖҒ*гҖҒ/гҖҒ==гҖӮ

е№ҝж’ӯ

е№ҝж’ӯпјҲBroadcastingпјүжҳҜдёҖз§ҚејәеӨ§зҡ„жңәеҲ¶пјҢе®ғе…Ғи®ё PyTorch еңЁжү§иЎҢз®—жңҜиҝҗз®—ж—¶еӨ„зҗҶеҪўзҠ¶дёҚеҗҢзҡ„ж•°з»„гҖӮ

е°ҶдёӨдёӘеј йҮҸиҝӣиЎҢе№ҝж’ӯпјҲbroadcastingпјүж—¶пјҢйҒөеҫӘд»ҘдёӢи§„еҲҷпјҡ

  1. еҰӮжһңдёӨдёӘеј йҮҸзҡ„йҳ¶пјҲrankпјүдёҚеҗҢпјҢе…ҲеңЁиҫғдҪҺйҳ¶еј йҮҸзҡ„еҪўзҠ¶пјҲshapeпјүеүҚйқўиЎҘ 1пјҢзӣҙеҲ°дёӨдёӘеҪўзҠ¶й•ҝеәҰзӣёеҗҢгҖӮ

  2. еҰӮжһңеңЁжҹҗдёӘз»ҙеәҰдёҠпјҢдёӨиҖ…зҡ„еӨ§е°ҸзӣёеҗҢпјҢжҲ–иҖ…е…¶дёӯдёҖдёӘеј йҮҸеңЁиҜҘз»ҙеәҰдёҠзҡ„еӨ§е°Ҹдёә 1пјҢйӮЈд№ҲиҝҷдёӨдёӘеј йҮҸеңЁиҝҷдёӘз»ҙеәҰдёҠиў«и®ӨдёәжҳҜе…је®№зҡ„гҖӮ

  3. еҸӘжңүеҪ“дёӨдёӘеј йҮҸеңЁжүҖжңүз»ҙеәҰдёҠйғҪе…је®№ж—¶пјҢе®ғд»¬жүҚиғҪиҝӣиЎҢе№ҝж’ӯгҖӮ

  4. е№ҝж’ӯд№ӢеҗҺпјҢжҜҸдёӘеј йҮҸзҡ„еҪўзҠ¶жҳҜдёӨдёӘиҫ“е…Ҙеј йҮҸеҪўзҠ¶еңЁйҖҗдёӘз»ҙеәҰдёҠзҡ„жңҖеӨ§еҖјгҖӮ

  5. еңЁд»»дҪ•дёҖдёӘз»ҙеәҰдёҠпјҢеҰӮжһңдёҖдёӘеј йҮҸзҡ„еӨ§е°ҸжҳҜ 1пјҢеҸҰдёҖдёӘеј йҮҸзҡ„еӨ§е°ҸеӨ§дәҺ 1пјҢйӮЈд№ҲиҝҷдёӘеӨ§е°Ҹдёә 1 зҡ„еј йҮҸеңЁиҜҘз»ҙеәҰдёҠиЎЁзҺ°еҫ—еғҸжҳҜиў«еӨҚеҲ¶дәҶдёҖж ·гҖӮ

е…¶д»–

йҡҗејҸзұ»еһӢиҪ¬жҚў

еёғе°”еј йҮҸдёҺжө®зӮ№ж•°гҖҒж•ҙж•°иҝҗз®—ж—¶иҮӘеҠЁиҪ¬жҚўдёә 0 е’Ң 1гҖӮ

ж•ҙж•°е’Ңжө®зӮ№ж•°иҝҗз®—ж—¶,з»“жһңдёәжө®зӮ№ж•°гҖӮдёҚеҗҢзІҫеәҰжө®зӮ№ж•°иҝҗз®—ж—¶,з»“жһңдёәжӣҙй«ҳзІҫеәҰгҖӮ

и®ҫеӨҮ

и®ӯз»ғиҝҮзЁӢдёӯжЁЎеһӢе’Ңж•°жҚ®еҝ…йЎ»еңЁеҗҢдёҖдёӘи®ҫеӨҮдёҠпјҢеҗҰеҲҷдјҡжҠҘй”ҷгҖӮ
